## Detailed Description

**UoPeople Sync** is a powerful productivity extension built for University of the People (UoPeople) students. In a single click, it deep-scans your course page and delivers a fully organized study toolkit.

---

### 🔍 What It Does

**1. Deep Course Scan**
Navigate to any UoPeople course page and press Scan. The extension automatically finds every Reading, Discussion, and Assignment — including their deadlines, learning outcomes, and linked resources — across all units.

**2. Obsidian Sync**
The scan results are instantly converted into a structured Markdown note and pushed to your Obsidian vault via the Local REST API plugin. Your note includes:
• A full **Course Schedule** table with deadlines
• A **Weekly Key Points Summary** (readings, discussions, assignments per unit)
• A clean list of **External Reading Links** for NotebookLM
• A list of **UoPeople Internal Files** for bulk download
• An AI-ready **NotebookLM Video Script Prompt**
• Full **Material Details** for every activity

**3. NotebookLM Integration**
Automatically builds a professional instructional-design prompt for Google NotebookLM. Paste it into your NotebookLM notebook to generate an 8–12 minute educational video script aligned to the unit's topics and learning outcomes.

**4. Copy Reading Links**
One-click copies all external reading URLs to your clipboard — ready to paste directly into NotebookLM as sources.

**5. Bulk File Download**
Downloads all UoPeople internal files (PDFs, PPTX, etc.) using your active login session, organized into a folder named after your course.

---

### 🔒 Privacy First

- Zero data sent to any external server.
- All processing happens locally in your browser.
- Only communicates with `my.uopeople.edu` (to read course content) and your local Obsidian instance (`127.0.0.1`).
- Your API key is stored securely in Chrome's local storage.

---

### ⚙️ Requirements

- An active UoPeople student account
- [Obsidian](https://obsidian.md/) desktop app with the **Local REST API** plugin installed and running

---

### 📌 How to Get Started

1. Install the extension.
2. Open the Obsidian Local REST API plugin and copy your API key.
3. Click the extension icon and paste your API key in Settings.
4. Navigate to any UoPeople course page.
5. Click **Deep Scan Course & Sync to Obsidian** — done!
