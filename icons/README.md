# Icons Directory

This folder should contain three PNG icon files for the Chrome extension:

| File          | Size       | Usage                             |
| :------------ | :--------- | :-------------------------------- |
| `icon16.png`  | 16×16 px   | Browser toolbar favicon           |
| `icon48.png`  | 48×48 px   | Chrome Extensions management page |
| `icon128.png` | 128×128 px | Chrome Web Store listing page     |

## Design Suggestion

- Background: Dark navy `#0f172a`
- Foreground: Graduation cap symbol or "U" monogram in `#93c5fd` (light blue)
- Style: Flat icon, no drop shadow, rounded corners on the icon canvas are handled by Chrome

## Free Icon Tools

- [Canva](https://canva.com) — free icon designer
- [favicon.io](https://favicon.io) — generate from emoji (use 🎓)
- [Figma](https://figma.com) — export at 16, 48, 128 px

## Quick Export with favicon.io

1. Go to https://favicon.io/emoji-favicons/graduation-cap/
2. Download the ZIP
3. Rename: `android-chrome-192x192.png` → `icon128.png`, etc.
4. Resize as needed with any image editor
